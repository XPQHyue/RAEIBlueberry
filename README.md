## 项目结构

| 目录/文件 | 说明 |
|-----------|------|
| `RA8P1/` | 瑞萨 RA8P1 嵌入式主控代码（e² studio 工程） |
| `RA8D1/` | RA8D1 Vision Board 开发板代码（蓝莓检测与定位） |
| `蓝莓控制源码.html` | 蓝莓采摘在线控制网页 |
| `蓝莓控制在线网站链接.txt` | 控制网页访问链接 |

## 技术栈

- **主控**: Renesas RA8P1（ARM Cortex-M85）
- **视觉**: RA8D1 Vision Board
- **控制**: 在线网页远程操控
- **机构**: 旋转式拾取装置
